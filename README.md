# Aventura - Fim dos Tempos

![foundry-v13-green](https://img.shields.io/badge/foundry-v13-green)

Bem-vindo ao módulo da Aventura de Fim dos Tempos feito para [Tormenta20](https://gitlab.com/vizael/Tormenta20/) para [Foundry Virtual Tabletop](https://foundryvtt.com/).

Aventura - Coração de Rubi é uma produção independente e não afiliada nem a [Jambô Editora](https://jamboeditora.com.br/) e nem ao grupo que fez o porte Não-oficial de Tormenta20 para o FoundryVTT. Para rodar a campanha, é necessário ter o livro da [Aventura - Fim dos Tempos (Arco 1)](https://jamboeditora.com.br/produto/jornada-heroica-fim-dos-tempos-arco-1/), além disso, é obviamente necessário que você tenha acesso ao [Livro Básico de Tormenta20](https://jamboeditora.com.br/produto/tormenta20-edicao-jogo-do-ano/).

Fim dos Tempos (Arco 1) é uma aventura baseado nas [streams](https://www.youtube.com/watch?v=KpUR4tjkcF0&list=PLvposoPkQY4IvKhAV0XixwYjuHV33NC12) onde Leonel Caldela prepara um conjunto de missões para desenvolver os personagens dentro das Colinas Centrais. Na minha opinião pessoal, é uma aventura bem focado no desenvolvimento dos NPC's e PC's com um combate desafiador no fim, é uma aventura que eu recomendaria para os novatos no sistema e no cenário. 

# O que tem nesse Compendium?

Esse Compendium visa ter os mapas, handouts, tokens e criaturas necessários para rodar a aventura, lembre-se, não iremos dar a história com esse módulo, apenas as coisas mais básicas do VTT para que você não tenha o trabalho e dor de cabeça de ter que fazer isso sozinho e na mão.

# O que NÃO tem nesse Compendium?

Esse módulo não tem nada de suplementos e muito menos contém a história da contida na aventura, perceba, apenas os contéudos desse módulo o deixarão perdido, é **necessário** você ter o livro da aventura para rodar a campanha.

## Instalação

### Instalação Semi-Automatica

1. Abra a página inicial do Foundry e navegue até a aba de módulos. 
2. Clique em "Intalar Novo Módulo".
3. Use o seguinte link de manifesto `https://github.com/mobguilherme/Aventura-Fim-dos-Tempos/releases/latest/download/module.json`
4. Clique "Instalar."

### Instalação Manual

1. Visite a [última versão](https://github.com/mobguilherme/Aventura-Fim-dos-Tempos/releases/download/v0.5.0/aventura-fim-dos-tempos.zip).
2. Faça o Download do arquivo "BestiariodeArton.zip".
3. Faça a descompactação do seu arquivo baixado na pasta de módulos do foundry, localizado no Data, caso precise de ajuda, visite: [FoundryVTT Configuration Guide](https://foundryvtt.com/article/configuration/#where-user-data '‌').
4. Reinicie o Foundry se necessário.
